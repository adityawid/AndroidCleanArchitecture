## Features

## Tech Stack
- MVVM (Model-View-ViewModel) Architecture Pattern
- Modularization (core module)
- Clean Architecture (data, domain, presentation)
- Dependency Injection with Dagger 2 & will be migrate to Hilt
- ViewBinding
- Room Persistence
- CameraX & MLkit
- Encryption with SQLCipher
